# Trabalho de Banco de Dados (BAN)

# Ambiente utilizado

* **Linguagem:** Java
* **Banco de Dados:** PostgreSQL
* **Driver JDBC:** postgresql-42.7.7.jar
* **IDE:** Eclipse

# Projeto:

Esse projeto tem como objetivo desempenhar as seguintes funcionalidades:

- Uma opção para inserir novas tuplas em todas as tabelas
- Uma opção para remover tuplas em todas as tabelas
- Uma opção para listar todas as tuplas de todas as tabelas
- Uma opção para listar o resultado de uma consulta que envolva uma junção entre duas tabelas
- Uma opção para listar o resultado de uma consulta que envolva subconsulta(s) e uma ou mais funções de agregação.

# Tabelas:

Baseadas no modelo conceitual e lógico entregue no Moodle:

* Cliente
* Conta
* Tipo Conta
* Agencia
* Transação
* Tipo transação

# Execução

* Crie o banco de dados com o script (BackUp) fornecido;
* Configure url, usuario e senha para a conexão com o banco.
  - Essas variáveis se encontram na pasta pesistenciaBD, na classe ConexaoBD;
* Com esses passos basta se dirigir a pasta Main, na classe PrincipalBD e executar o código.
* Após a execução será dadas todas as opções que atendem às diretrizes estipuladas no projeto.
